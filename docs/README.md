## How to generate the dashboard

```bash
# Export a completed run
optimizer dashboard --run-id <id> --no-server
# → generates docs/data.json

# Serve locally
optimizer dashboard --run-id <id> --open
# → http://localhost:8080

# Deploy to GitHub Pages
git add docs/data.json
git push
# → https://your-org.github.io/LoopBench-Optimizer/
```

## Adding a `?refresh=N` parameter

When serving locally during an active run, add `?refresh=5` to auto-refresh every 5 seconds:

```
http://localhost:8080?run_id=abc123&refresh=5
```
